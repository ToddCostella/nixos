tmux

prefix = Alt+a
Sessions
prefixtheno
Session switcher (sessionx + zoxide)
prefixthen$
Rename session
prefixthend
Detach from session
prefixthen( )
Prev / next session
tmux new -s name
New named session (shell)
tmux ls
List sessions (shell)
tmux attach -t name
Attach to session (shell)
sx
Session switcher alias (shell)
Windows
prefixthenc
New window (inherits current path)
prefixthen,
Rename window
prefixthen&
Close window (confirm)
Alt+1 – Alt+9
Jump to window by number (no prefix)
prefixthenn p
Next / prev window
prefixthenw
List / choose window
prefixthenl
Last (previously active) window
prefixthenf
Find window by name
Panes
prefixthen|
Split pane horizontally (current path)
prefixthen-
Split pane vertically (current path)
prefixthenx
Close pane (confirm)
Alt+←↑↓→
Navigate panes (no prefix)
prefixthen;
Toggle last active pane
prefixthenz
Zoom / unzoom pane (fullscreen)
prefixthen!
Break pane into new window
prefixthenq
Show pane numbers
prefixthen{ }
Swap pane left / right
prefixthenspace
Cycle pane layouts
Copy Mode (vi)
prefixthen[
Enter copy mode (scroll)
q or Esc
Exit copy mode
v
Begin selection
y
Yank selection → Wayland clipboard
prefixthen]
Paste buffer
/
Search forward
?
Search backward
n N
Next / prev search match
g G
Jump to top / bottom
Misc
prefixthen?
Show all key bindings
prefixthen:
Open command prompt
prefixthent
Show clock (24h)
prefixthen~
Show tmux messages
prefixthenM-a
Send prefix to inner session
mouse
Click pane / scroll / drag pane border
