# COSMIC Desktop Keyboard Shortcuts

The **Super** key (Windows/OS key) is the primary modifier for most actions.

## Window Navigation & Focus

| Shortcut | Action |
|----------|--------|
| `Super + Arrow Keys` | Move focus between windows |
| `Super + Tab` | Cycle through windows |
| `Shift + Super + Tab` | Cycle windows in reverse |

## Workspace Management

| Shortcut | Action |
|----------|--------|
| `Super + Ctrl + Arrow Keys` | Navigate between workspaces |
| `Super + 1-9` | Jump to workspace 1-9 |
| `Super + 0` | Switch to last workspace |

## Moving Windows

| Shortcut | Action |
|----------|--------|
| `Shift + Super + Arrow Keys` | Move window in tiling layout |
| `Shift + Super + Ctrl + Arrow Keys` | Move window to different workspace |
| `Shift + Super + 1-9` | Send window to workspace 1-9 |

## Window Control

| Shortcut | Action |
|----------|--------|
| `Super + Q` | Close window |
| `Super + M` | Maximize window |
| `Super + F11` | Fullscreen mode |
| `Super + R` / `Shift + R` | Resize window |
| `Super + S` | Toggle window tabs/stacking |

## Tiling Window Management

| Shortcut | Action |
|----------|--------|
| `Super + Y` | Enable/disable tiling for workspace |
| `Super + G` | Toggle floating mode for window |
| `Super + O` | Switch orientation (vertical ↔ horizontal) |
| `Super + X` | Enter swap mode (use arrows to swap windows) |
| `Super + U` | Expand selection in tile tree |
| `Super + I` | Collapse selection in tile tree |

## Application & System Access

| Shortcut | Action |
|----------|--------|
| `Super` | Open launcher/search |
| `Super + A` | Applications library |
| `Super + W` | Workspaces overview |
| `Super + F` | Open Files |
| `Super + T` | Open Terminal |
| `Super + Escape` | Lock screen |
| `Super + Shift + Escape` | Log out |
| `Print` | Screenshot |

## Accessibility/Zoom

| Shortcut | Action |
|----------|--------|
| `Super + =/-` | Zoom in/out |
| `Super + .` / `Super + ,` | Zoom in/out (alternate) |

---

All shortcuts can be customized in **COSMIC Settings → Input Devices → Keyboard → Keyboard Shortcuts**.
